About This Site

What is this?

THIS IS NOT AN OFFICIAL MATERIA MAGICA WEBSITE.

This is a fan-made equipment database for Materia Magica. All information comes from my personal database and player submissions.

What's Working (50% chance)

  • Searching items
  • Items showing correct color (mostly)
  • Item page showing locations, mobs, and room names
  • Mob pages with spawn locations and dropped items
  • MUSHclient plugin for capturing equipment
  • Log importer — pulls gear from MUD logs
  • Database importer — imports your .db files

How It Works

Items are pulled from two main sources: the in-game EQDB (Equipment Database) and personal player submissions via the MUSHclient plugin. EQDB items appear with stats as recorded by the game, while personal items are tagged as such.

The MUSHclient plugin captures equipment data as you identify items in-game. It saves them locally while you play and uploads to the database when triggered. This catches things the EQDB might not have, but it can also pick up extra text or miss some fields.

Accuracy varies because Materia Magica's internal DB records every new instance it encounters. If someone drops a random item in a room, it gets recorded as belonging there. If a mob is found somewhere new, it gets logged there too. Summoned creatures, elementals, and roaming mobs end up listed in every room they've ever been seen in. This means locations, spawns, and drops can be misleading — always verify in-game before relying on any data here.

Want to try the plugin? Hit me up (Huko) in-game. It's early in development.
